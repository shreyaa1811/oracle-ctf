SYNTHETIX AI — INFORMATION SECURITY POLICY
Version 2.1 | Classification: Internal

All production systems must use MFA. Passwords must be minimum 16 characters.
Data at rest must be encrypted using AES-256. Data in transit requires TLS 1.3 minimum.
Access to production databases is restricted to the Infrastructure team and requires a change request ticket.
Penetration testing is conducted bi-annually by an approved third party.
Employees must complete security awareness training annually.
Report security incidents to: security@synthetix-ai.internal within 1 hour of detection.
